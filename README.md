# Fluxer themes

I made some bs

How to use:
1. goto the link to raw css file (example: `https://raw.githubusercontent.com/GibberishDev/fluxer-themes/refs/heads/main/sharp.css`) or go to theme file on github and click `raw`
2. select everything and paste in fluxer's custom css overrides (settings > look and feel > Theme > Custom theme tokens > Custom theme overrides) ~~I would use @import css statement but i fint it unstable with current build of fluxer~~
3. Done
